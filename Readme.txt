Digital Library is a web application which is implemented using Java Servlet Pages (JSP). It allows the admin to search, add, issue, return and remove books. Additionally it also allows admin to add students who can issue and return books. There is also a student portal which students can access and perform various operations like issuing and returning a book.

Set up your desktop

Pull the git repository. 
Go to netbeans and open the project and run web-> main1.jsp
